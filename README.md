## myreads
my reads is a bookshelf ap that allows you to select and categorize books you have read, are currently reading, or want to read.

## Quickstart
#### Install
To get Started:
* change to the project directory `cd reactnd-project-myreads`
* install all project dependencies with `npm install`
* start the server with `npm start`
